# OpenCV-Playground-Interface
Websocket Server and Client for sending live updates to OpenCV-Playground.

---

## Inital Setup
1. Clone repo
2. `git submodule init`
3. `git submodule update`
4. `sh init.sh`

---

## Serve Static
`sh serve.sh`

__or__

From /OpenCV-Playground:
- `npm run build`

And from /server:
- `npm run serve`

---

## Run Dev Server
From /OpenCV-Playground:
- `npm run dev`
